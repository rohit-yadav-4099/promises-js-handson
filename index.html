Q.6- Create examples to explain callback hell function

Ans:

Example: 
        function fun1(res){
            console.log('1');
            res();
        }
        function fun2(res){
            console.log('2')
            res();
        }
        function fun3(res){
            console.log('3')
            res();
        }
        
        
        fun1(()=>{
            fun2(()=>{
                fun3()
            })
        })
Output: 
        1
        2
        3

Explanation: In this question, multiple function call inside another function .It is known as pyramid problem or triangle problem.

Q.8- Create examples to explain async await function

Ans:

Example: 
        let promise = new Promise(function (resolve, reject) {
            setTimeout(function () {
                resolve('hello')}, 1000); 
            });
        async function display() {
            let result = await promise; 
            console.log(result);
            console.log('Welcome');
        }
        display();
Output: 
        hello
        Welcome

Explanation: In this example use the async await function, a Promise object is created and it gets resolved after 1000 milliseconds. Here, the display() function is written using the async function. the await keyword waits for the promise to be complete (resolve or reject). and generate the output.

Q.9- Create examples to explain promise.all function

Ans:

Example: 
        const promise1 = Promise.resolve(50);
        const promise2 = 500;
        const promise3 = "Welcome";
        const promise4 = new Promise((resolve) => {
            setTimeout(() => {
                resolve("hello");
                }, 1000);
            });
        
        Promise.all([promise1, promise2, promise3, promise4]).then((values) => {
            console.log(values);
        });
Output: 
        [ 50, 500, 'Welcome', 'hello' ]

Explanation: The Promise.all() method is one of the promise concurrency methods / static methods. use for aggregating the results of multiple promises. In this example use four promises and promises as input abd returns a single promise. returned promise fulfills when all promises fulfill and generate the output in array form.
